== Description ==
Brozzme Lightbox Blurb Module in Divi add a new module when using the page builder.
The magic comes in front-end.

There's no option to set for this plugin.

This plugin is made for Divi Theme 3.0+

[Benoti](https://brozzme.com/ "Brozzme") and [WPServeur](https://www.wpserveur.net/?refwps=221 "WPServeur WordPress Hosting").

== Installation ==
1. Upload the plugin on wordpress.org and copy the archive to the "/wp-content/plugins/" directory.
2. Activate the plugin through the "Plugins" menu in WordPress.

= What else ? =
